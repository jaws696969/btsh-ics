BTSH Team Calendars

Use the calendar URL below to subscribe in Google Calendar, Apple Calendar, Outlook, or another calendar app. Clicking the raw ICS link may download the file instead of creating a live subscription.

How to subscribe

  1. Google Calendar: Copy the calendar URL, then in Google Calendar click the + next to “Other calendars” and choose From URL.
  2. Apple Calendar: Click the webcal button, or use File → New Calendar Subscription and paste the URL.
  3. Outlook: Add a subscribed calendar from the web / internet calendar and paste the URL.
